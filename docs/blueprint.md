# **App Name**: Cierre de Caja

## Core Features:

- Input Sales Data: Allows users to input the different sales channels and amounts for each (cash, cards, transfers, gift cards, and deliveries from different vendors like Pedidos Ya Ice Scroll, Pedidos Ya Wafix, Pedidos Ya Mix, Uber Eats, and Junaeb).
- Calculate Total Sales: Automatically calculates the total sales for the day by summing up the amounts entered for all the different sales channels.
- Calculate Expected Cash Balance: Takes the starting cash balance, adds the total cash sales, and subtracts any cash expenses to determine the expected cash balance.
- Generate Daily Report: Create an executive summary of all sales channels

## Style Guidelines:

- Primary color: Warm green (#8FBC8F) to symbolize financial transactions in a friendly way.
- Background color: Light gray (#F0F0F0), nearly white but with slight visual texture.
- Accent color: Soft gold (#D4AF37) for highlighting totals and important figures.
- Body text and headlines: 'PT Sans' for a modern but clear presentation of numeric data.
- Use clear, consistent icons to represent different payment methods and delivery services.
- A clean, tabular layout for easy data entry and readability.
- Subtle transitions to highlight calculations.
